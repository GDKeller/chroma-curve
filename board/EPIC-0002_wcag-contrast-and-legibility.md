# WCAG Contrast & Legibility

Multiple functional UI labels and controls fail WCAG AA contrast requirements. Control labels use `text-white/20` (~1.4:1) where 4.5:1 is required for small text. Compact swatch labels render at 7–9px which is below any legibility threshold.

## Scope

- Raise contrast on all functional labels across Header, PaletteGrid, SaturationCurve, EffectiveSaturation, About (UX-5, UX-6)
- Fix or hide compact swatch labels that render below 8px (UX-7)
