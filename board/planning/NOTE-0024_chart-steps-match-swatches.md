# Sync chart step dots with swatch count

When the swatch count is reduced (e.g. 5), the curve charts still show dots for every lightness step. Add an option to toggle between showing steps for all lightness values or just the colors in the palette. When showing all values, visually distinguish swatch-aligned dots from the rest (e.g. larger/brighter dots for swatch steps, dimmed or hidden for others).
